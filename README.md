DN Classic Auto Dance

# How to setup
1. Download the source code
2. Install python
3. Install requirements by running `pip install -r requirements.txt`.
4. Run `python main.py`
5. Set the UI size to Stage 4
6. Start playing dance event.
7. On game region, hover the mouse on the upper left of the box where arrows appear (add some allowance for 8 keys) then press F8. Next is the lower right then press F8.
- Check 
9. On timing region, same as in the game region but start in the upper left of the timing bar (from blue)
